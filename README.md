

# 轻舟云主机系统MMUI模板 MMUI Template for Qzsystem

## 特性

- **毛玻璃模式**：现代而典雅，提供半透明效果，增强视觉体验。
- **正常模式**：标准的亮色界面，清晰简洁。
- **黑暗模式**：护眼的暗色主题，适合在低光环境下使用。

## 安装方法

1. 下载模板文件。
2. 将下载的文件解压，找到`index`文件。
3. 将此`index`文件复制到您的网站目录`/www/wwwroot/你的网站/view/control/ecs`中，替换原有文件。
